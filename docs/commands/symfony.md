## Scripts and console commands




### cache clear

```bash
bin/console cache:clear
```


### bin/console list

```bash
bin/console list
```


### bin/console routes

```bash
bin/console debug:router
```


### asset install
```
bin/console assets:install
```

### Dump autoload

```
cd /vagrant/www/projects/opendevops
sudo rm -rf vendor/composer/autoload_* && composer dump-autoload
```
